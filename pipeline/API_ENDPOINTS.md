# API Endpoints map-making.app

## ✅ Endpoints confirmés et utilisés

Tous les endpoints utilisent la base URL: `https://map-making.app`

### 1. Test de connexion / Auth

**GET** `/api/user`

```bash
curl -H "Authorization: API {apiKey}" \
     https://map-making.app/api/user
```

**Response:**
```json
{
  "id": "user_uuid",
  "name": "john_doe",
  "email": "user@example.com"
}
```

---

### 2. Créer une nouvelle map

**POST** `/api/maps`

```bash
curl -X POST \
     -H "Authorization: API {apiKey}" \
     -H "Content-Type: application/json" \
     -d '{"name": "La ligue", "description": ""}' \
     https://map-making.app/api/maps
```

**Response:**
```json
{
  "id": "map_uuid_123",
  "name": "La ligue",
  "description": "",
  "publicUrl": null
}
```

### 6. ⭐ Importer les coordonnées (PRINCIPAL)

**POST** `/api/maps/{mapId}/locations`

```bash
curl -X POST \
     -H "Authorization: API {apiKey}" \
     -H "Content-Type: application/json" \
     -d '{
       "edits": [
         {
           "action": {"type": 4},
           "create": [
             {
               "id": -1,
               "flags": 0,
               "location": {
                 "lat": 48.8566,
                 "lng": 2.3522
               },
               "panoId": null,
               "heading": 45.0,
               "pitch": -10.0,
               "zoom": 0.5,
               "tags": ["S5", "J0", "Solo", "Move", "link=xyz"],
               "createdAt": "2026-03-19T16:30:00.000Z"
             }
           ],
           "remove": []
         }
       ]
     }' \
     https://map-making.app/api/maps/map_uuid_123/locations
```

**Details:**
- `action.type: 4` = command to create locations
- `id: -1` = auto-generated by server
- `flags: 0` = no special flags
- `location: {lat, lng}` = GPS coordinates
- `panoId: null` = no Google Street View panorama
- `tags: [...]` = Array of tag strings
- `createdAt: ISO8601Z` = Creation timestamp

---

### 7. Récupérer les coordonnées (protobuf - binaire)

**GET** `/api/maps/{mapId}/locations`

```bash
curl -H "Authorization: API {apiKey}" \
     -H "Accept: application/protobuf" \
     https://map-making.app/api/maps/map_uuid_123/locations \
     --output locations.pb
```

⚠️ Format protobuf binaire - pas utilisé dans cette pipeline

## 🔐 Authentification

Toutes les requêtes utilisent le header:

```
Authorization: API {apiKey}
```

Où `{apiKey}` est la clé générée sur https://map-making.app/keys

---

## 📊 Batch Import (recommandé for >500 coords)

La pipeline divise automatiquement en chunks si > 500 coordonnées:

```python
# Python (dans la pipeline)
uploader.batch_import_by_chunks(
    map_id="map_uuid_123",
    coordinates=large_list,
    chunk_size=500
)
```

Chaque chunk = 500 coordonnées max en une requête pour éviter les timeouts.

## ✗ Endpoints NOT utilisés / NOT recommandés

❌ **GeoGuessr API** (`https://www.geoguessr.com/api/v4/*`)
- Pas besoin pour cette pipeline
- Nécessite authentification GeoGuessr
- Différent endpoint

❌ **Endpoints de draft** 
- `/api/v4/user-maps/drafts/{draftId}` → GeoGuessr only
- `/api/v4/user-maps/maps/{mapId}` → GeoGuessr only

---

## 📝 Notes

- Tous les endpoints retournent du JSON (sauf `/locations` GET qui retourne protobuf)
- Timeouts: ~30s pour requête standard, ~60s pour import bulk
- Rate limiting: Pas de limite connue, mais utiliser du bon sens
- Max taile par requête: ~100MB
